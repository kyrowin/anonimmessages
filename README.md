## How to use
install python 3.12

**install git**
$ sudo apt install git

**clone repository**
$ git clone https://github.com/kyrowin/anonimmessages.git

**open folder**
$ cd anonimmessages

**install libs**
$ pip install -r requirements.txt

**add bot token and your user id to bot.py**

**start bot**
$ python bot.py

enjoy!

# commands
/start - start bot
/broadcast (text) - broadcast text to all bot users
